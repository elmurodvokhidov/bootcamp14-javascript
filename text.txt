document.write() - ushbu metod yordamida javascript dan turib html documentimizga ma'lumotlarni yozishimiz mumkin.
Comparison (solishtirish) - Ikki operandni bir-biri bilan solishtiradi va bunday operatorlar true yoki false qiymatini qaytaradi (ya'ni boolean)

== - type ni hisobga olmagan holda ikki operandning tengligini solishtiradi
=== - type va qiymati bo'yicha ikki operandning tengligini solishtiradi
!= - ikki operandning teng emasligini tekshiradi (not type)
!== - type ni ham hisobga olgan holda, ikki operandning teng emasligini tekshiradi
> - birinchi qiymat ikkinchisidan katta bo'lsa true, aks holda false qaytaradi
< - birinchi qiymat ikkinchisidan kichik bo'lsa true, aks holda false qaytaradi
>= - birinchi qiymat ikkinchisidan katta yoki teng bo'lsa true, aks holda false qaytaradi
<= - birinchi qiymat ikkinchisidan kichik yoki teng bo'lsa true qaytaradi aks holda false

Boolean Logic (mantiqiy operatorlar) - Dasturlar o'zlarida ma'lum bir solishtirishlar orqali qaror qabul qiladi. Boolean Logic algebraning bir qismi hisoblanib, qiymatlar true (rost) yoki false (yolg'on) ko'rinishida bo'ladi. Boolean Logic asosiy uch operatordan iborat. Bular: AND, OR va NOT
Boolean Logic 'ni JS 'da Logical Operators sifatida ham ishlatish imkoniyati mavjud.

AND (va) - operator ko'rinishi quyidagicha &&. Agar barcha qiymatlar true dan tashkil topgan bo'lsa, natijaviy qiymat true bo'ladi. Aks holda false

false && false = false            0 * 0 = 0
false && true = false             0 * 1 = 0
true && false = false             1 * 0 = 0
true && true = true               1 * 1 = 1

OR (yoki) - operator ko'rinishi quyidagicha ||. Agar kamida bir qiymat true dan tashkil topgan bo'lsa, u holda natijaviy qiymat true bo'ladi, aks holda false

false || false = false            0 + 0 = 0
false || true = true              0 + 1 = 1
true || false = true              1 + 0 = 1
true || true = true               1 + 1 = 1

NOT - operator ko'rinishi quyidagicha !. Berilgan qiymat true bo'lsa false, false bo'lsa true qiymati yuzaga keladi

!false = true 
!true = false
